# SlotsFreeUSA Visual QA Checklist

Last updated: 2026-05-28

Use this checklist before public promotion, affiliate outreach follow-up, ad-network applications, or live partner links.

Do not add live ads, affiliate links, tracking scripts, payment tools, credentials, API keys, or public AI during visual QA.

## Test Devices / Viewports

Check at minimum:

- Mobile narrow width: 360px to 390px
- Mobile large width: 414px to 430px
- Tablet width: 768px
- Desktop width: 1280px or wider

If testing manually, use Chrome or Edge DevTools device toolbar.

## Priority Pages

Test these first:

1. `index.html`
2. `play-free-slots.html`
3. `free-slots.html`
4. `no-download-slots.html`
5. `search.html`
6. `reviews.html`
7. `how-we-review.html`
8. `partner-with-us.html`
9. `contact.html`
10. `responsible-gaming.html`
11. `privacy.html`
12. `terms.html`
13. `affiliate-disclosure.html`

## Header / Navigation Checks

For each priority page:

- Header is visible.
- Logo links to `index.html`.
- Menu works on mobile.
- Menu can close after a link is tapped.
- Header links are readable and not cramped.
- No horizontal scrolling on mobile.
- Current navigation does not hide important pages.

## Footer Checks

For each priority page:

- Footer is visible.
- Footer links are readable on mobile.
- Footer reaches key trust pages:
  - About
  - Play Free
  - Free Slots
  - Slot Terms
  - Free Spins
  - Bonuses
  - Reviews
  - How We Review
  - Partner With Us
  - Contact
  - Privacy
  - Terms
  - Disclosure
- Footer does not contain duplicate nav blocks.

## Homepage Checks

Check:

- Main headline is clear within first screen.
- 21+ / no-gambling disclosure is visible.
- Free-play call to action is obvious.
- Lucky Lou section does not crowd the page.
- Review/trust cards are readable.
- Newsletter or signup placeholder does not collect data if inactive.
- Page feels like a finished website, not a placeholder.

## Free Slot Game Checks

On `play-free-slots.html`:

- Reels display correctly.
- Spin button works.
- Reset button works.
- Virtual credit count updates.
- Last win updates.
- Message text changes after spin.
- Game cannot double-spin rapidly.
- No real-money language appears.
- No deposits, payouts, or prizes are implied.
- Layout works on mobile without clipping.

## Search Page Checks

On `search.html`:

- Search input is visible and usable on mobile.
- Search button is easy to tap.
- Search for `slots` returns useful pages.
- Search for `bonus` returns bonus/free-spin pages.
- Search for `social` returns social/sweepstakes pages.
- No-result message is helpful.
- Search does not require a server or external tool.

## Trust / Partner Checks

On review and partner pages:

- No live partner buttons unless approved.
- Pending/approved/paused/removed language is clear.
- Partner inquiry path points to `partner-with-us.html` or `contact.html`.
- Affiliate disclosure is easy to find.
- Responsible-play language is visible.
- No guaranteed-win, easy-money, or legal-everywhere claims appear.

## Legal / Safety Checks

Check all legal/trust pages:

- `privacy.html` states current privacy posture.
- `terms.html` says the site is informational and does not operate gambling.
- `affiliate-disclosure.html` explains future compensation clearly.
- `responsible-gaming.html` includes stop signs and safer-play reminders.
- `contact.html` uses `slotsfreeusa@gmail.com`.

## Mobile Readability Checks

For all priority pages:

- Text is readable without zooming.
- Buttons are easy to tap.
- Tables do not break mobile layout.
- Cards stack cleanly.
- No content is cut off.
- No accidental horizontal scroll.
- Sections have enough spacing.

## Performance / Simplicity Checks

Confirm:

- No heavy external scripts.
- No ad scripts.
- No tracking scripts.
- No payment scripts.
- No public AI widget.
- Images and SVGs load.
- Pages feel fast.
