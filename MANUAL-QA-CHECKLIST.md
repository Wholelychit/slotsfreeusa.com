# SlotsFreeUSA Manual QA Checklist

Last updated: 2026-05-28

Use this checklist before affiliate outreach, ad-network review, or public promotion.

## 1. Homepage Check

- [ ] `https://slotsfreeusa.com/` loads without errors.
- [ ] The top message clearly says the site is for adults 21+.
- [ ] The page says SlotsFreeUSA does not offer real-money gambling.
- [ ] The main button opens the free slot page.
- [ ] The review cards do not use live affiliate links unless approved.
- [ ] The newsletter area does not collect emails until consent setup is ready.

## 2. Navigation Check

Open the menu and test every main link:

- [ ] Home
- [ ] Play Free
- [ ] Free Slots
- [ ] Bonuses
- [ ] Reviews
- [ ] Social Casinos
- [ ] Responsible Play
- [ ] Contact
- [ ] Privacy
- [ ] Terms
- [ ] Disclosure

## 3. Mobile Check

Test on a phone-size screen:

- [ ] Header fits.
- [ ] Menu opens and closes.
- [ ] Buttons are easy to tap.
- [ ] Cards stack cleanly.
- [ ] Footer links are readable.
- [ ] No text runs off the screen.

## 4. Trust Page Check

These pages should load and make sense without ChatGPT explaining anything:

- [ ] `affiliate-disclosure.html`
- [ ] `responsible-gaming.html`
- [ ] `privacy.html`
- [ ] `terms.html`
- [ ] `contact.html`
- [ ] `how-we-review.html`
- [ ] `partner-with-us.html`

## 5. SEO Guide Check

These public guides should load, have one clear H1, include a canonical URL, and avoid live affiliate links unless approved:

- [ ] `free-slots.html`
- [ ] `play-free-slots.html`
- [ ] `best-free-slots-for-beginners.html`
- [ ] `no-download-slots.html`
- [ ] `slot-terms-explained.html`
- [ ] `free-spins-explained.html`
- [ ] `casino-bonus-terms-explained.html`
- [ ] `what-are-social-casinos.html`
- [ ] `sweepstakes-casino-guide.html`

## 6. Contact Check

- [ ] Contact page shows `slotsfreeusa@gmail.com`.
- [ ] The email link opens a mail app.
- [ ] The page warns visitors not to send passwords, IDs, payment details, or private account data.
- [ ] Partner inquiry instructions are clear.

## 7. Crawl Check

- [ ] `robots.txt` loads.
- [ ] `robots.txt` allows public crawling.
- [ ] `robots.txt` points to the sitemap.
- [ ] `sitemap.xml` loads.
- [ ] Sitemap includes all major public pages, including `no-download-slots.html`.
- [ ] Every sitemap page exists.
- [ ] Every sitemap page has a matching canonical URL.
- [ ] Public pages do not include accidental `noindex` tags.

## 8. Search Check

- [ ] `search.html` loads.
- [ ] Searching `free slots` returns relevant pages.
- [ ] Searching `no download` returns `no-download-slots.html`.
- [ ] Searching `bonus terms` returns bonus and free-spin guides.
- [ ] Empty or unmatched searches show readable fallback content.

## 9. Affiliate Readiness Check

Before applying to a partner program:

- [ ] Homepage is live.
- [ ] Contact email is visible.
- [ ] Review hub is live.
- [ ] How We Review page is live.
- [ ] Affiliate Disclosure page is live.
- [ ] Responsible Play page is live.
- [ ] At least five helpful guide pages are live.
- [ ] No placeholder partner links are presented as real offers.
- [ ] No unsupported legal-availability claims appear.
- [ ] No earnings or winnings are promised.

## 10. Compliance Language Check

Remove or rewrite any phrase like:

- guaranteed wins
- risk-free gambling
- easy money
- beat the casino
- available everywhere
- legal in every state
- best payout

Use safer wording:

- play for fun
- read the terms
- rules vary by site and location
- no guaranteed winnings
- we do not accept bets or deposits

## 11. Speed Check

- [ ] Images are not oversized.
- [ ] No autoplay video exists.
- [ ] No unnecessary third-party scripts exist.
- [ ] No live ads are installed before approval.
- [ ] No live tracking scripts are installed before privacy setup.

## 12. Final Review

Only promote or send affiliate applications after the site feels understandable to:

1. A first-time visitor.
2. Google.
3. An affiliate manager.
4. An ad network reviewer.
5. Gerry reviewing it from a different computer.
