# SlotsFreeUSA Style Guide

## Brand Personality
SlotsFreeUSA should feel:
- clear
- trustworthy
- mobile-first
- easy to use
- adult-focused
- beginner friendly
- visually clean

## Writing Style
Use:
- short paragraphs
- simple wording
- calm explanations
- direct language
- educational tone

Avoid:
- hype
- exaggerated promises
- fake urgency
- confusing casino slang
- unrealistic winning claims

## Visual Style
Preferred colors:
- dark navy backgrounds
- gold highlights
- white/light gray text

## Layout Rules
- large buttons
- readable spacing
- mobile-first sections
- simple cards
- limited clutter

## SEO Rules
Every public page should include:
- unique title
- meta description
- internal links
- readable headings
- footer navigation

## Accessibility
- readable text sizes
- strong contrast
- simple navigation
- mobile-friendly buttons

## Monetization Rules
Ads and partner links should:
- remain clearly separated
- avoid deceptive placement
- remain readable
- not overwhelm content
