## Required Review Fields

Every future review should include:

- Partner name
- Partner type
- Status
- Best for
- Age note
- Location note
- Offer summary
- Important terms
- Responsible-play note
- Disclosure note
- Last reviewed date
- Link status

## Copy Rules

Use simple language.

Do not use:

- guaranteed wins
- risk-free gambling
- easy money
- legal everywhere
- best payout
- beat the casino
- no risk

Use safer language:

- terms vary
- read official rules
- play for entertainment
- no guaranteed winnings
- location rules may apply
- set limits
- take breaks
